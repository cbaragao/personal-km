# Template: Basic Note

Use this template to create a new note.

```markdown
---
layout: default
title: Your Note Title
parent: Notes  # Or the appropriate parent category
tags: [tag1, tag2, tag3]
date: YYYY-MM-DD
---

# Your Note Title

Brief introduction or summary of what this note is about.

## Main Content

Your content here. You can use Markdown formatting:

- Bullet points
- For lists

1. Numbered lists
2. Work too

### Subsections

Use headings to organize your thoughts.

## Related Items

- [Link to related note](../path/to/note)
- [External resource](https://example.com)

## Tags

#tag1 #tag2 #tag3
```
